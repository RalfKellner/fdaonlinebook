Introduction to Python#

Python is an interpreted programming language. This means that code written by a human is translated by the interpreter so that the computer understands the human’s instructions. The Python Interpreter can be downloaded for free from the official homepage. However, to work pleasantly and efficiently with Python, it is recommended to install Python together with a development environment. Popular examples are Anaconda, VS Code or PyCharm. However, you do not need to install any of these options as we will be working using notebooks that can be opened and run directly in Google Drive using Google Colab. The only requirement is a Google account. If you do not have a Google account and do not want to create one, please contact me via email.

What follows is a small introduction to Python, which should help you better understand code examples from the course. The goal towards the end of the course should also be that you can adapt and evaluate existing code examples for your own analyses. I therefore recommend that you actively participate in learning the content from the beginning. The best way to do this is to open an empty Colab Notebook on your Google Drive in parallel with this material and reproduce the code examples. In the best case, experiment a bit with the examples by making slight changes and check for yourself if the output is what you expect. To run a programmable cell, click in the cell and run the code by clicking on the play icon. Alternatively, you can click in the cell and press Control+Enter on your keyboard. This will execute all the code in the cell. Only lines preceded by the # symbol will be ignored by the interpreter, since this is a comment.

Python as a calculator#

In its simplest form, Python can be used like a calculator. The elementary mathematical functions are:

  • + for addition

  • - for subtraction

  • * for multiplication

  • \ for division

  • ** for the exponential calculation

Addition

print("Addition:")
print(2 + 2)

print("Subtraction:")
print(42 - 21)

print("Multiplication:")
print(2 * 2)

print("Division:")
print(5 / 2)

print("Exponential calculation:")
print(2**3)

Other arithmetic operations are:

  • \ for the floor division

  • % for the modulus

In addition, of course, the usual rules of mathematics apply, such as the commutative, associative and distributive laws.

Functions in Python#

As with common calculators, “build-in” functions exist for Python. Examples are the minimum or maximum operator or the absolute value function. Function names are always written in lower case. The arguments of the function are enclosed in parentheses after the function name.

print("The maximum of the numbers 2, 4, 5 is:")
print(max(2, 4, 5))

print("The minimum of the numbers -1, 2, 4 is")
print(min(-1, 2, 4))

print("Der absoulte value of the number -2 is:")
print(abs(-2))

Object assignments in Python#

An option that makes analysis very easy is the assignment of intermediate results using the = symbol. In this way, results can be saved during the analysis and used for further analysis. There are only a few restrictions when assigning the object name. For example, the object names must not begin with a number. In addition, there are recommendations for naming standards, but these are not mandatory. Nevertheless, adherence to these or at least a consistent type of naming increases the clarity and understanding of the code.

a = 2
b = 3
c = a**3
print(c)

Data structuring#

Often a dataset consists of multiple observations of different variables such as the age, weight, and hair length of multiple people. We will see very practical and advanced ways to temporarily store and analyze such data using Python throughout the course. As a foundation, we will first look at the fundamental constructs in Python to collect multiple data points:

  • lists

  • tuples

  • Dictionaries

To create the desired object in each case, the symbols:

  • List: []

  • Tuple: ()

  • Dictionary: {}

are used. Lists and tuples are a collection of individual data or objects. The main difference is the mutability in lists and the immutability in tuples. Individual values from lists and tuples can be accessed with the correct index (position in the list or tuple). Dictionaries are similar to traditional dictionaries and consist of key-value pairs. Individual values are accessed by identifying them with the matching key. For a better understanding, let’s consider some examples in the cell below.

# define a list
my_list = [1, 2, 3]

# define a tuple
my_tuple = (1, 2, 3)

# define a dictionary
my_dict = {}
my_dict['Eins'] = 1
my_dict['Zwei'] = 2
my_dict['Drei'] = 3

# to access single values in a list or tuple, one must use the matching index
# IMPORTANT: in Python, counting starts at 0
print(my_list[0])

print(my_tuple[1])

print(my_dict['Drei'])

1
2
3

# to access parts from lists and tuples, one needs to use slicing
# access the first to values
print(my_list[:2])

# access the values from 1 to 3
print(my_list[1:3])

# with the - symbol, we start slicing from the end of the list or tuple
print(my_tuple[-2:])

[1, 2]
[2, 3]
(2, 3)

There are many other ways to extract more complex parts of a list or a tuple. For reasons of the clarity, we want to do without these for the time being and/or fall back with possible uses on it. For the time being, it should be mentioned that lists and tuples can not only contain single numbers, but also objects as single elements. For example, a list can consist of several lists. If you only want the unique values from a list or a tuple and not the duplicates, the set exists for this purpose.

# a nested list
nested_list = [[1, 2], [3, 4]]
print(nested_list[0])

# the set excluding duplicated values
list_with_duplicates = [1, 2, 2, 4, 4, 5]
print(set(list_with_duplicates))

[1, 2]
{1, 2, 4, 5}

Define own functions#

Often analyses have to be performed in an individual way or in a way that is specific to the problem. In particular if certain parts of the analysis are repetitively accomplished, it offers itself to define own functions. This possibility exists in Python where a function is a coherent sequence of code commands to be interpreted. In Python, a function is initiated by the def keyword, followed by a self-selected name for the function and a bracket in which function arguments are defined. The beginning of the code sequence starts and ends with indentation. If a function is defined, nothing is evaluated for the time being. This happens only with the call of a function. It is sufficient if a function is defined once at the beginning. It is best to look at a few examples.

# a function without an argument
def say_hello():
    print("Hello!")

# a function with argument
def add_one(number):
    return number + 1

# a function with a list as an argument which returns the maximum and the index of its value
def find_maximum(a_list):
    max_value = max(a_list)
    max_index = a_list.index(max_value)
    return max_value, max_index    

# let's execute those functions
say_hello()

print(add_one(2))

print(find_maximum([1, 2, 3]))

Hello!
3
(3, 2)

Which data types are passed as arguments can usually be found out by calling help(function name). Basically all data types can be passed to functions and can also be executed by functions. However, if an error occurs during the execution of the function, no result is passed, but an error message. Advanced aspects on the subject of functions are the use of type hinting and the passing of any number of function arguments.

Modules and packages#

If it is convenient for a project to define multiple functions, they can be defined once in a separate file and imported for different analyses. For example, in my current folder I have stored a file named my_module.py, in which the following functions are defined:

def print_python_is_fun():
    print("Python is fun to learn!")

def add_two(number):
    return number + 2

Functions from this file can now be imported in the following way:

# Either using the from ... import ... way
from my_module import print_python_is_fun, add_two

print_python_is_fun()
add_two(2)

Python is fun to learn!

4

# we can also import all functions using the * symbol, this is rather uncommon
from my_module import *

print_python_is_fun()
add_two(2)

Python is fun to learn!

4

# often modules are imported with an alias and methods are accessed combining the alias with the dot and the function name
import my_module as mm

mm.print_python_is_fun()
mm.add_two(2)

Python is fun to learn!

4

The file is called module and has the advantage that functions or also objects can be defined there, which can be used for several analyses after import again and again. If you go one step further and bundle several modules thematically in a folder, you get what is called a package in Python. This is one of the greatest strengths of Python. There are many packages written by developers and Python users that are freely available and cover a wide range of modern methods in computer science, mathematics, statistics and data analysis. As an example you can see in the code below how to import one function from each module of the package my_package.

from my_package.your_module import print_something_else
from my_package.my_module import print_python_is_fun

print_python_is_fun()
print_something_else()

Repeated evaluations for elements of an object can be executed using the for loop.

# a simple loop
people = ["Andrew", "Max", "Francis"]
for person in people:
    print(person)
print("-"*50)

# a loop using a running index
for i in range(5):
    print(i)
print("-"*50)

# a look using index enumeration and an object
for i, person in enumerate(people):
    print(f"Person number {i+1}: {person}")
print("-"*50)

# a loop over two objects
people = ["Andrew", "Max", "Francis"]
ages = [21, 24, 23]
for age, person in zip(ages, people):
    print(f"{person} is {age} years old.")
print("-"*50)

# a loop using index enumeration and two objects
people = ["Andrew", "Max", "Francis"]
ages = [21, 24, 23]
for i, (age, person) in enumerate(zip(ages, people)):
